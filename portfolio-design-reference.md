# Portfolio Design Reference

## Style: Dark Modern + Warm Accent
**Category:** Professional Developer Portfolio

---

## Typography
- **Font:** Inter (Google Fonts)

## Sections

### 1. HERO
- Particle animation (floating orange dots)
- Photo with stacked cards effect
- Gradient text on name
- Two CTA buttons (gradient + outlined)
- Bouncing scroll-down indicator
- Highlighted keyword in accent color

### 2. ABOUT
- Two-column: image left, text right
- Photo with border/frame
- Sub-headings: "My Journey" + "My Approach"
- 3 stat cards with icons

### 3. EXPERIENCE (Timeline)
- Vertical timeline with orange line + dots
- Cards on alternating sides
- Badge pill (e.g. "internship")
- Company logo + accent-colored name
- Location + date meta
- Tech stack pills + bullet points

### 4. SKILLS
- 6 category cards (3x2 grid)
- Gradient icon boxes
- Skill pills/tags inside cards
- Hover effects

### 5. PROJECTS
- 3-column project card grid
- Screenshot + title + description
- Colored tech stack pills
- Action links (Live Demo, Watch, Try Demo)
- Card hover lift effect

### 6. RESUME
- Info card with document icon
- 3 action buttons (Hide/Download/Open in New Tab)
- Embedded PDF preview (iframe/embed)
- Toggle show/hide

### 7. CONTACT
- Two-column: info left, form right
- Orange icon squares for contact info
- Labeled inputs, dark background, rounded
- Gradient submit button with send icon

### 8. FOOTER
- 3-column: Brand / Links / Social
- Social icons row
- Darker background
